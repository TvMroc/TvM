Ik ben Thomas. En ik zit in het eerste jaar van Sofware Developer bij het ROC.

Over mij

Ik ben Thomas van Meelis en ik woon in lent. Ik heb met computers werken altijd al leuk gevonden daarom heb ik voor de opleiding Software Developer gekozen. Ik ben ook best goed in engels omdat mijn vader en zijn kant van mijn familie engels spreekt. Ik kan ook rubiks kubussen oplossen, dit heb ik toen ik 10 was op de basis school geleerd bij creatief wiskunde. Vroeger had ik mee gedaan met een lego robot webstrijd op de het ROC de school waar ik nu op zit. je moest de robot programmeren om langs verschillende obstakels te gaan wie dit het beste had gedaan won.

Studie

Ik doe de opleiding software developer bij het ROC, bij deze opleiding leer je hoe je software kan ontwikkelen voor verschillende soorten apparaten. Op de opleiding krijg ik les over Website, Programmeren, ICT en Tools. Bij Website leer ik over HTML, CSS en JS maar ook over web design. Bij de Programmeer lessen gaat het over Java, python of andere talen. Bij ICT lessen gaat het over veel verschillende dingen bijvoorbeeld: Cyber security, Internet of Things, Computer onderdelen en hoe computers werken. bij Tools leer ik over verschillende apps zoals Git, Docker, Office en IMovie.

Wat ik kan

De codeer talen die ik op dit moment ken zijn: HTML, CSS, JS en Java. HTML is de taal die ik het beste ken, want die is ook het makkelijkst. CSS is de taal die ik gebruik om de stijl van een website aan te passen. ik gebruik vaak Flexbox om mijn elementen te positioneren. JS is de taal die ik het minst heb gebruikt, maar ik heb er wel al wat mee gedaan. Met JS kan ik op dit moment dingen maken zoals dropdowns en selectie. Java is de taal die ik gebruik om software te maken voor computers. Een voorbeeld van een programma dat ik heb gemaakt met Java is het spel Mastermind. Mastermind is een spel waarbij je een code moet raden die bestaat uit gekleurde pinnen. Je kunt de settings van het spel instellen, zoals het aantal kleuren dat in het spel zit, het aantal pinnen in de code en hoevaak je mag raden.
