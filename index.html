Over mij

Ik ben Thomas van Meelis, ik woon in lent en mijn favoriete kleur is blauw. Ik heb met computers werken altijd al leuk gevonden daarom heb ik voor de opleiding Software Developer gekozen. Ik ben ook best goed in engels omdat mijn vader en zijn kant van mijn familie engels spreekt. Ik kan ook rubiks kubussen oplossen, dit heb ik toen ik 10 was op de basis school geleerd bij creatief wiskunde. Wat ook best cool is dat ik vroeger mee had gedaan met een lego robot webstrijd op de school waar ik nu op zit. je moest de robot programmeren om langs verschillende obstakels te gaan wie dit het beste had gedaan won.

Studie

Als je de opleiding software developer volgt, word je opgeleid tot een professional die software kan ontwikkelen voor verschillende soorten apparaten. Je kunt bijvoorbeeld software maken voor computers, telefoons, tablets, smartwatches of robots. Je leert niet alleen hoe je software programmeert, maar ook hoe je software ontwerpt en test. Je leert ook hoe je goed kunt communiceren en samenwerken met klanten, collegaâ€™s en andere betrokkenen. De opleiding software developer is een mbo-opleiding op niveau 4, wat betekent dat je een hoog niveau van kennis en vaardigheden hebt. De opleiding duurt 3 of 4 jaar, afhankelijk van je vooropleiding en je leerweg. Je moet ook goed zijn in rekenen, Nederlands en Engels, want dat zijn belangrijke vakken voor een software developer. En je moet interesse hebben in computers en techniek, want dat is waar je mee bezig bent.

Wat ik kan

De codeer talen die ik op dit moment ken zijn: HTML, CSS, JS en Java. HTML is de taal die ik het beste ken, want die is ook het makkelijkst. CSS is de taal die ik gebruik om de stijl van een website aan te passen. ik gebruik vaak Flexbox om mijn elementen te positioneren. JS is de taal die ik het minst heb gebruikt, maar ik heb er wel al wat mee gedaan. Met JS kan ik op dit moment dingen maken zoals dropdowns en selectie. Java is de taal die ik gebruik om software te maken voor computers. Een voorbeeld van een programma dat ik heb gemaakt met Java is het spel Mastermind. Mastermind is een spel waarbij je een code moet raden die bestaat uit gekleurde pinnen. Je kunt de settings van het spel instellen, zoals het aantal kleuren dat in het spel zit, het aantal pinnen in de code en hoevaak je mag raden.
